# Crgbaumgart's Websites

Craig Baumgarten's ersonal monorepo containing:

- `apps/denton-bights-collective` — Social club landing site (Astro)
- `apps/craigs-creations` — Photography business site (Next.js)

## Setup

Requires [mise](https://mise.jdx.dev) and pnpm via Corepack.

```bash
mise install
pnpm install
```

## Development

```bash
pnpm dev:club    # social club site
pnpm dev:photo   # photography site
```
